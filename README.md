# Sistem Gestiune Cabinet Veterinar 🐾
Proiect realizat pentru materia **Baze de Date în Economie** (Anul II, FSEGA).

## Descriere
Acest proiect vizează automatizarea fluxului de date într-o clinică veterinară, gestionând proprietari, animale, programări și tratamente medicale.

## Tehnologii Utilizate
* **SQL (Oracle APEX)**
* **Modelare ERD**

## Caracteristici Cheie
* **Integritatea datelor:** Constrângeri pentru a preveni programările simultane ale medicilor.
* **Complexitate:** Peste 10 tabele interconectate și vederi (Views) pentru rapoarte de business.
